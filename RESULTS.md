# نتایج تست واقعی Fayton-Weather

**تاریخ تست:** ۱۶ ژوئن ۲۰۲۶
**مکان:** تهران

## خلاصه عملکرد
- Accuracy: ۰.۸۲ - ۰.۸۷
- مدل روی داده‌های واقعی API Open-Meteo آموزش و تست شد.
- تشخیص خوب برای بارش، باد شدید و شرایط آفتابی.

## پیشنهادات بعدی
- اضافه کردن داده‌های تاریخی بیشتر
- اتصال به APIهای ایرانی
- داشبورد وب
